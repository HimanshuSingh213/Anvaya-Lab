# Color Style

BACKGROUND

-   background → Whole application background
-   surface → Main panels, sidebar, cards
-   surface-secondary → Nested panels, tabs, inputs
-   surface-elevated → Modals, popovers, dropdowns
-   surface-hover → Hover state
-   surface-active → Selected/active state

TEXT

-   foreground → Primary text
-   foreground-secondary → Labels, metadata
-   foreground-muted → Helper text, placeholders
-   foreground-disabled → Disabled text

BORDERS

-   border → Default borders
-   border-hover → Hovered borders
-   border-active → Focused/selected borders

BRAND

-   accent → Primary actions, links, active indicators
-   accent-hover → Hover state for accent

STATUS

-   success → Success messages
-   warning → Warning states
-   danger → Errors & destructive actions
-   info → Informational messages

HTTP METHODS

-   method-get → GET badge
-   method-post → POST badge
-   method-put → PUT badge
-   method-patch → PATCH badge
-   method-delete → DELETE badge
-   method-head → HEAD badge
-   method-options → OPTIONS badge

MISC

-   ring → Focus outline
-   overlay → Modal/drawer backdrop
-   selection → Text selection highlight
